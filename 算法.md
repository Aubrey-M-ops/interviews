# 算法

#### 数组

- Map 
- 双指针
- 遇到比大小或求和，先排序

#### 链表

- 合并、删除（指定k快慢指针）
- 反转(pre cur next循环里定义)
- 环（flag）

#### 栈和队列

- 栈

  - 括号

  - 维护递减队列

- 队列

  - 栈转为队列
  - <u>双端队列</u>

#### DFS BFS

- DFS:维护栈 二叉树前序遍历 (根右左)
- BFS：维护队列 二叉树逐层打印时维护队列

#### 二叉树

- 二叉搜索树
  - 右>=根 左<=根
  - 查找 插入 删除
- 平衡二叉树

#### 排序

- 基础
  - 冒泡 两两比较，大的放后面
  - 插入 当前元素前面序列有序，往前面插
  - 选择 最小的放最前面
- 进阶排序
  - 归并
  - 快排
